## Installation

The easiest way to install CrabKeeper is to use the [`cargo`](https://crates.io/crates/cargo) package manager.

```
cargo install crabkeeper
```

## Usage

```
crabkeeper <command>
```

For a list of commands, run `crabkeeper --help`.

```
crabkeeper --help
```
